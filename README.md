This mini-game was developed for the computer graphics course @cs.uoi.gr. Made in Unity.<br />
Unity Version: 2020.3.25f1
## Controls
- Move: W, A, S, D, Q, E
- Shoot: Space
- Camera: Mouse
## Build and Run
You can build and play this game using the Unity Hub or you can unzip the game build included in the repository and run MeteorShower.exe.
